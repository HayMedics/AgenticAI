CardioRisk AI — Coronary Artery Disease Risk Prediction System
A research-grade clinical decision support system for Coronary Artery Disease (CAD) risk classification, built to TRIPOD-AI reporting standards and deployed as a public web application. It predicts CAD probability from 13 clinical features using a calibrated stacking ensemble (Gradient Boosting + Logistic Regression + Random Forest + Extra Trees), with isotonic calibration and a cost-sensitive threshold optimised to minimise missed diagnoses (FN penalty = 10x FP).
Model performance: AUC-ROC 0.87; Sensitivity 100% (zero false negatives on test set); NPV 100%; decision threshold 0.18 (cost-optimised).
Key features: SHAP-style feature explanations per prediction, downloadable clinical PDF reports, a What-If Simulator with clinical plausibility checks, confidence intervals via Monte Carlo perturbation, population distribution comparison (n=302), batch CSV processing, an external validation page (TRIPOD-AI Item 12), and session-based prediction history.
Key discovery: Identified and corrected an inverted target label bug in the widely-used Kaggle version of the Cleveland dataset — documented publicly in the project README.
Tech stack: Python, scikit-learn, Streamlit, ReportLab, pandas, matplotlib, GitHub, Streamlit Cloud.
Standards: TRIPOD-AI, ACC/AHA 2019, ATP III, Google Model Card framework.
Live demo: https://cardioriskaihaymedics.streamlit.app (Research use only. Not validated for clinical decision-making.)

StrokeGuard AI — Clinical Stroke Risk Prediction Web App
An end-to-end machine learning web app that estimates a patient's stroke risk from routine clinical inputs (age, blood pressure history, glucose, BMI, smoking status and more). Designed, built, and deployed from scratch to demonstrate responsible, explainable ML in healthcare.
Highlights: a leak-free scikit-learn pipeline (imputation, scaling, encoding) with a calibrated logistic regression model; handled severe class imbalance (only ~5% of patients had a stroke) using class weighting; chose a cost-based decision threshold that weights a missed stroke 10x a false alarm, prioritising sensitivity as a screening tool should; added explainability (SHAP-style feature contributions), Monte-Carlo confidence intervals, a what-if simulator, population comparison, batch CSV screening, session history, a model-validation page, and branded PDF reports; documented in the spirit of the TRIPOD-AI reporting standard and deployed live on Streamlit Cloud.
Honest performance on a held-out test set: ROC-AUC approximately 0.84 with high negative predictive value. It is an educational screening tool, not a medical device.
Live app: https://strokeguardaihaymedics.streamlit.app
Code: https://github.com/HayMedics/strokeguard-ai
